# CRUD TODO Django

## Init Project
```
mkdir TODO

cd TODO

python3 -m venv env

source env/bin/activate

pip install Django

django-admin startproject todoapp

cd todoapp

django-admin startapp tasks
```

## Run Dev Server
```
python manage.py runserver
```
